## project_english
This project is a simple Python program that calculates the average grade of a student.   It is a basic algorithm designed to demonstrate how variables, inputs, and conditions work in Python.  The program is a beginner-friendly example for students learning programming.


## How it works
1. The program asks the user to enter the student's name.
2. The program asks for three grades.
3. The algorithm calculates the average by adding the grades and dividing the result by three.
4. The program checks if the average is greater than or equal to 3.0.
5. The system prints whether the student passes or does not pass the course.
